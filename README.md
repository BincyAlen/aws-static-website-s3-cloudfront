# AWS Static Website Hosting using S3 and CloudFront

## Overview
This project demonstrates hosting a static website on Amazon S3 and distributing it globally using Amazon CloudFront with HTTPS.

## AWS Services Used
- Amazon S3
- Amazon CloudFront
- AWS IAM
- AWS Budgets

## Key Features
- Static website hosting
- Global content delivery via CDN
- HTTPS enabled using CloudFront default certificate
- Secure and cost-optimized AWS setup
